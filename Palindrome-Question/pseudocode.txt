Input  - a phrase or a word to check if it can be read same in both backward and forward ways
1. Create a an input for a phrase
2. Convert the input to lower case
3. Remove all the spaces and specical characters within that phrase
5. Reverse the clean phrase
6. If the cleaned phrase is equal with the reversed phrase we say the phrase if a palindrome and can be read both backwards and forward.
7. Else we say the phrase is not a palindrome and can't be read both backwards and forward at the same time
Output  - 'true' if the word/phrase can be read same in backward and forward ways and
        'false' it it can't be read the same
